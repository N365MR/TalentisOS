# TalentisOS QA Checklist

## Phase 0 Foundation

- `index.html` opens directly in a browser.
- `styles.css` loads from a relative path.
- `app.js` loads from a relative path.
- `manifest.json` is linked from `index.html`.
- `service-worker.js` exists and registers only in supported contexts.
- The page displays `TalentisOS`.
- The page displays `Your Daily Leadership Playbook`.
- No console errors appear during a normal local run.
- No frameworks, backend, or package manager are required.
- The project is ready for Phase 1.
